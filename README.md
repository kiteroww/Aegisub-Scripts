# Kiterow Aegisub Scripts

A collection of Aegisub automation scripts for timing, quality control, typesetting, gradients, and color management.

## Scripts

| Script | Version | Menu root | Interface languages |
| --- | --- | --- | --- |
| Chrono Suite | 1.0.0 | `Chrono Suite` | English, Spanish, Portuguese |
| Rhea Signs | 1.0.0 | `Rhea Signs` | English, Spanish, Portuguese |
| Gradient Row | 1.6.0 | `Gradient Row` | English |
| AddTexture | 2.0.1 | `AddTexture` | English |
| Zheus Colormanager | 4.2.0 | `Zheus Colormanager` | Spanish |

## Installation

1. Install [DependencyControl](https://github.com/TypesettingTools/DependencyControl).
2. Copy the files from `Macros/` into the Aegisub autoload directory.
3. Restart Aegisub or reload the Automation scripts.

Windows autoload directory:

```text
%APPDATA%\Aegisub\automation\autoload\
```

DependencyControl update information is provided in `DependencyControl.json`.
